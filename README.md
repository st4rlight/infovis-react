# 伪基站垃圾短信的可视化
## 技术栈

前端框架：`React`  `使用(create-react-app)构建`

  图标库：`iconfont`  `fontawesome`

  请求库：`axios`

    ui库：`antd`

  分词库：`结巴分词node.js实现  node-jieba`

  图表库：`echarts` 、 `echarts-for-react` 、`echarts-wordcloud`

    地图：`高德地图 amap-js` 、`react-amap`、`react-amap-plugin-heatmap`

    其他：`csvToJson`
